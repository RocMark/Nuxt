# Nuxt Template

## 💿 Install 
```bash
# install
npm i

# install & generate .env file (will overwrite)
npm run init
```


## 🚀 Usage
```bash
# serve with hot reload at localhost:3001
npm run dev

# build for production and launch server ( For SSR )
npm run build
npm run start

# generate static project ( Use on GithubPage )
npm run generate
```
